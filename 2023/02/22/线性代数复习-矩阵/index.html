1.矩阵的基本运算

    矩阵的乘法不要死记，把它看作线性组合或线性变换。

$$\begin{pmatrix}b_1 \ b_2 \\dots \b_n\end{pmatrix}A= \begin{pmatrix}b_1A \ b_2A \\dots \b_nA\end{pmatrix}$$,$b_1\dots b_n$是行向量

2.矩阵运算的性质

    需要注意的是：

  1. 矩阵乘法不满足交换律即AB=BA不一定成立，甚至可能一个有意义而另外一个都不能乘。

  2. 矩阵乘法也不满足消去律，即AB=AC不一定能推出B=C（A列满秩时可以），AB=O也不能推出A=O（B行满秩时可以）或B=O（A列满秩时可以）。

    2的证明

    法一: 设$B=(b_1,b_2 \dots b_n)$,$AB=(Ab_1\dots Ab_n)=0$,所以$Ab_i=0$, 又因为$A$列满秩,所以$AX=0$只有零解，所以$b_i=0$

    法二: (写成分块矩阵) 设$PA=\begin{pmatrix} E_n \ O\end{pmatrix},PAB=\begin{pmatrix} E_n \ O\end{pmatrix}B=\begin{pmatrix} B \ O\end{pmatrix}=O$,所以$B=O$

矩阵乘法的可交换性

已知$A$,求$AB=BA$的矩阵$B$.

一般做法:设$B=(b_{ij})_{m\times n}$,利用矩阵乘法的定义列出方程

四、矩阵的秩*

矩阵的秩的定义

  1. 非零子式的最高阶数，定义零矩阵的秩为0
  2. 把矩阵化作（行）阶梯矩阵后阶梯头的个数（相抵标准形中单位矩阵的阶数）
  3. 矩阵行/列向量组的极大线性无关组中向量的个数。

​

行/列满秩的定义$A_{m \times n}$

若$r(A)=n,n \leq m$,则称$A$列满秩 ，相抵标准型为$\begin{pmatrix} E_r \ O\end{pmatrix}$

若$r(A)=m,m \leq n$,则称$A$行满秩 ，相抵标准型为$\begin{pmatrix} E_r \ O\end{pmatrix}$

证明的工具

  1. 行列式的定义、性质
  2. （分块）矩阵初等变换
  3. 相抵标准型
  4. 向量组的线性表示、向量组的秩、线性方程组的解空间

常用的秩的公式及其证明*

$\max(r(A),r(B))\leq r(A\ B) \leq r(A)+r(B)$. $r(\begin{matrix}A \ B \end{matrix})\leq r(A)+r(B)$

这里是分块矩阵的记号

左边: A的k阶非零子式也是(A B)的非零子式，故$r(A \ B)\geq r(A)$

右边: 对矩阵做初等变换

$r(A)=r(A^T)=r(AA^T)=r(A^TA)$ !!!!!

$r(A)=r(A^T)$用行列式的定义显然

$AX=O$能推出$A^T(AX)=O$

$A^TAX=O$能推出$X^TA^TAX=O=(AX)^T(AX)$,进而推出$AX=O$

两方程组同解，由线性方程组解空间的结论，可推出秩相等

$C=\begin{pmatrix}A \ O \ O \ B\end{pmatrix}$或$\begin{pmatrix}O \ A \ B \ O\end{pmatrix}$, $r(C)=r(A)+r(B)$

证明: 对A,B分别做初等变换，最终C化为两个相抵标准型

$D=\begin{pmatrix}A \ C \ O \ B\end{pmatrix}$或$\begin{pmatrix}C \ A \ B \ O\end{pmatrix}$, $r(D)\geq r(A)+r(B)$

法二:

$r(AB) \leq \min{r(A),r(B)}$

法二（相抵标准型）见书

$r(A+B) \leq r(A)+r(B)$

法一(向量组的秩) $A=(\alpha_1 \dots \alpha_n),B=(\beta_1\dots \beta_n)$

$A+B$的列向量组是$(\alpha_1+\beta_1 \dots \alpha_n+\beta_n)$,一定能被$(\alpha_1 \dots \alpha_n\ \beta_1\dots \beta_n)$线性表示。因此$r(A+B) \leq r(A\ B) \leq r(A)+r(B)$(用到结论1)

再由结论1，左边$\geq r(A+B \ \ B)\geq r(A+B)$

若A列满秩,则r(AB)=r(B). 若B行满秩，则r(AB)=r(A)

六、一些特殊类型的矩阵

零矩阵不一定是方阵，所以不一定对称

单位矩阵、对角矩阵、数量矩阵、准对角矩阵

单位矩阵:主对角线上元素为1，其余元素均为0的方阵，记作E（可加下标n）；

对角矩阵:除主对角线以外元素均为0的n阶方阵，记作diag(k1,k2,…,kn)；对角矩阵和同阶对角矩阵可交换

数量矩阵：主对角线元素都相等的对角矩阵，相当于kE；数量矩阵与任意同阶矩阵可交换

上/下三角矩阵：

主对角线下方元素全为0的矩阵称为上三角矩阵，记作U；主对角线上方方元素全为0的矩阵称为下三角矩阵，记作L；这两类矩阵统称为三角矩阵，任意矩阵可以分解为这两类矩阵的乘积，即A的LU分解；

对称矩阵*

对称矩阵:$A=A^T$

定义相关:

  • 对称矩阵的和、数乘、幂仍然是对称矩阵

  • 对任意矩阵，$AA^T,A^TA,A+A^T$是对称矩阵。且$r(AA^T)=r(A^TA)=r(A)$

  • 如果$A^T=A,A^2=O$,那么$A=O$。 证明：$A^2=AA^T=O$,乘积对角线元素为$\sum_{j=1}^na_{ij}^2=0$.

  • 若A,B是对称矩阵，则$AB$是对称矩阵当且仅当AB=BA。 证明: $(AB)^T=B^TA^T=BA=AB$ (因此AB不一定对称的)

实对称矩阵相关:

  • 实对称矩阵的特征值一定为实数，特征向量为实向量 (见特征值一章的证明)
  • 不同特征值特征向量相互正交
  • 实对称矩阵必可相似对角化

反对称矩阵*

反对称矩阵: $A=-A^T$

  • 反对称矩阵的和、数乘是反对称矩阵。奇数次幂是反对称矩阵，偶数次幂是对称矩阵。

  • 若A,B是反对称矩阵，则$AB$是反对称矩阵当且仅当AB=BA ，证明同对称矩阵

  • 若A,B是反对称矩阵，则矩阵$AB-BA$是反对称矩阵

  • 奇数阶反对称矩阵对应的行列式的值必为0 （推论:如果可逆，那一定是偶数阶的证明:$|A|=|A^T|=|-A|=(-1)^n|A|$

  • $(A^*)^T=(A^T)^*=(-A)^*=(-1)^{n-1} A^*$. 所以n为偶数时伴随矩阵反对称，否则对称

  • A为反对称矩阵的充要条件是，对任意n维列向量X,$X^TAX=0$.

    证明: 必要性，$X^TAX=-X^TA^TX=-(X^TAX)^T=-X^TAX$

    ​ 充分性, 取$X=(0 … 1…0)$,得$a_{ii}=0$. 再取$X=(0…1..0..1..1)$(第i个和第j个为1，其他为0)。 $X^TAX=a_{ii}+a_{ij}+a_{ji}+a_{jj}=a_{ij}+a_{ji}=0$。 所以$a_{ij}=-a_{ji}$

  • 反对称矩阵的秩必为偶数。

  • 实反对称矩阵的特征值一定是纯虚数或0，证明同实对称矩阵的特征值为实数

任意一个方阵都可以唯一表示为对称矩阵和反对称矩阵的和

存在性:$A=\frac{1}{2}(A+A^T)+\frac{1}{2}(A-A^T)$

唯一性: 若$A=B_1+C_1=B_2+C_2$,转置得$B_1-C_1=B_2-C_2$.两式相加得$B_1=B_2,C_1=C_2$.
