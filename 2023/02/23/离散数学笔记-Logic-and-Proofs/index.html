1.3 Propositional Equivalences

Tautology: Compound propositions that are always true( e.g. p∨¬pp \vee\neg pp∨¬p)

Contradiction: Compound propositions that are always false

Compound propositions that have the same truth values in all cases are called logically equivalent. p≡qp \equiv qp≡q

Logical Equivalences

Using truth tables, we can prove these equivalences:

p∨0=p∧1=pp \vee 0=p \wedge 1=pp∨0=p∧1=p

p∨1=1,p∧0=0p \vee 1=1,p \wedge 0=0p∨1=1,p∧0=0

Satisfiability

A compound proposition is satisfiable if there is an assignment of truth values to its variables that makes it true

Normal Form

Literal: ppp or ¬p\neg p¬p

Distributive Normal Form (DNF) : A formula is said to be in disjunctive normal form if it is written as a disjunction, in which all the terms are conjunctions of literals(a variable or its negation). etc. (p∧¬q)∨r(p \wedge \neg q)\vee r(p∧¬q)∨r. We can also define Conjunctive Normal Form(CNF)

A minterm is a conjunction of literals in which each variable is represented exactly once. Forp,q,rp,q,rp,q,r p∧qp \wedge qp∧q s not a minterm. p∧q∧rp \wedge q \wedge rp∧q∧r is a minterm.

2n2^n2n minterms.

Transforming into Full DNF:First ∧(p∨¬p)\wedge(p \vee \neg p)∧(p∨¬p), then use distributive law

Full DNF can be used to construct a proposition with a given truth table. etc. proposition consisting of 3 variables p,q,r is true when exactly two variables are true. (p∧q∧¬r)∨(p∧¬q∧r)∨(¬p∧q∧r)(p \wedge q\wedge \neg r)\vee(p \wedge \neg q\wedge r)\vee(\neg p \wedge q\wedge r)(p∧q∧¬r)∨(p∧¬q∧r)∨(¬p∧q∧r). Therefore, every compound proposition is logically equivalent to a full DNF.

1.4 Predicates and Quantifiers

Predicates

A predicate (propositional function) is a statement that contains variables. Once the values of the variables are specified, the function has a truth value.

P(x1,x2…xn)P(x_1,x_2\dots x_n)P(x1​,x2​…xn​) is called n-ary or n-place predicate. such as P(x)=x>0P(x)= x>0P(x)=x>0

Quantifiers

Universal Quantifier: ∀x p(x)\forall x\ p(x)∀x p(x)
Existential Quantifier: ∃x p(x)\exist x\ p(x)∃x p(x)

Domain (domain of discourse / universe of discourse): range of the possible values of the variable x

Precedence: ∀\forall∀ and ∃\exists∃ have higher precedence than all logical operators

∀xP(x)∨Q(x)≡(∀xP(x))∨Q(x)\forall x P(x) \vee Q(x) \equiv (\forall x P(x)) \vee Q(x)∀xP(x)∨Q(x)≡(∀xP(x))∨Q(x)

Binding Variables

Bound variable: a variable is bound if it is quantified.

All the variables in a propositional function must be quantified or set equal to a particular value to turn it into a proposition.
