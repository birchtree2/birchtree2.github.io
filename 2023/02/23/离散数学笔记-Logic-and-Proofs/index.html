离散数学笔记-Logic and Proofs

1.Propositional Logic

开学前预习一下
ZJU离散数学的教材是全英文的， Discrete Mathematics and Its Applications, Kenneth H. Rosen
要注意很多名词的写法

Propositions Logic

Propositions is a statement that is either true or false,but not both

Logic Operators

  1. Negation: ¬p\neg p¬p

  2. Conjunction(and) p∧qp \wedge qp∧q

  3. Disjunction(or) p∨qp \vee qp∨q

  4. Implication(If p then q / p implies q / p only if q) p→qp \to qp→q

    注意p only if q 也是 $p \to q$，相当于必要条件

    Truth Table:

    p q p ->q
    1 1 1
    1 0 0
    0 1 1
    0 0 1

    ​ Everything implies true. False implies everything

  5. Biconditional : p↔qp \leftrightarrow qp↔q

Propositional Equivalences

Tautology: Compound propositions that are always true( e.g. p∨¬pp \vee\neg pp∨¬p)

Contradiction: Compound propositions that are always false

Compound propositions that have the same truth values in all cases are called logically equivalent. p≡qp \equiv qp≡q

Logical Equivalences

Using truth tables, we can prove these equivalences:

p∨0=p∧1=pp \vee 0=p \wedge 1=pp∨0=p∧1=p

p∨1=1,p∧0=0p \vee 1=1,p \wedge 0=0p∨1=1,p∧0=0

De Morgan’s Laws: \neg(p \and q)=\neg p \or \neg q,\neg(p \or q)=\neg p \and \neg q

Distribution: p \and(q \or r)=(p \and q) \or(p \and r)

​ p \or(q\and r)=(p \or q)\and (p \or r)

Absorption Laws: p \or(p \and q)=p, p \and(p \or q)=p

Conditional: p \to q \equiv \neg p \or q (Because p implies q is false only if p=1 and q=0)

Then using these laws we can prove:

Conditional statements:

p→q≡¬q→¬pp \to q \equiv \neg q \to \neg pp→q≡¬q→¬p

Biconditional statements:

p \leftrightarrow q \equiv (p \to q)\and(q\to p)

p↔q≡¬p↔¬qp \leftrightarrow q \equiv \neg p \leftrightarrow \neg qp↔q≡¬p↔¬q

p \leftrightarrow q \equiv (p \and q)\or(\neg p \and \neg q)

第三条从真值表来看是显然的。但是也可以用上面的定律证,多次用分配律拆括号,然后用p \and \neg p=0的性质

Satisfiability

A compound proposition is satisfiable if there is an assignment of truth values to its variables that makes it true
